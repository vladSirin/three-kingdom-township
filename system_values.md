# 三国小镇 · 系统数值设计文档

> **版本**: v1.0 · 2026-03-04
> **用途**: 集中管理所有游戏内隐藏数值，方便策划迭代。前端 UI 只显示模糊描述，玩家不可见具体数字。

---

## 1. 资源系统

### 1.1 资源列表

| 资源 | 图标 | 初始值 | 固定范围 | 上限计算 | 游戏结束条件 |
|------|------|--------|----------|----------|-------------|
| 民心 (morale) | 👥 | 50 | 0 – 100 | **固定 100** | ≤ 0 或 ≥ 100 |
| 粮草 (food) | 🌾 | 30 | 0 – cap | 40 + 粮仓等级 × 20 | ≤ 0 或 ≥ 100 |
| 武力 (military) | ⚔️ | 30 | 0 – cap | 40 + 军械库等级 × 20 | ≤ 0 或 ≥ 100 |
| 财富 (wealth) | 💰 | 30 | 0 – cap | 40 + 官署等级 × 20 | ≤ 0 或 ≥ 100 |
| 声望 (reputation) | 📜 | 50 | 0 – 100 | **固定 100** | ≤ 0 或 ≥ 100 |

### 1.2 前端描述映射

> 玩家只看到模糊描述，不直接可见数值。

| 百分比区间 (value/cap) | 描述词 | 进度条颜色 |
|----------------------|--------|-----------|
| 0% – 10% | 崩溃 | 🔴 深红闪烁 |
| 11% – 25% | 危急 | 🔴 红色 |
| 26% – 40% | 低迷 | 🟠 橙色 |
| 41% – 60% | 平稳 | 🟡 金色 |
| 61% – 80% | 兴旺 | 🟢 绿色 |
| 81% – 90% | 鼎盛 | 🔵 蓝色 |
| 91% – 100% | 过盛 | 🟣 紫色闪烁 |

> [!IMPORTANT]
> **过盛 (≥91%)** 同样危险！资源达到 100 会触发 Game Over。紫色闪烁用于警告玩家。

## 2. 建筑系统

### 2.1 建筑等级与效果

| 建筑 | 图标 | 最大等级 | 数值效果 | 前端描述 (替代公式) |
|------|------|---------|---------|-------------------|
| 城墙 (wall) | 🏯 | 5 | 防御等级 = Lv | Lv1 "勉强抵挡" → Lv3 "坚固可靠" → Lv5 "固若金汤" |
| 农田 (farm) | 🌾 | 5 | 产粮 = Lv × (人口/20) | Lv1 "荒芜薄田" → Lv3 "沃土千亩" → Lv5 "天府之国" |
| 市场 (market) | 🏪 | 5 | 产金 = Lv × (人口/40) | Lv1 "零星小贩" → Lv3 "商贾云集" → Lv5 "万商之都" |
| 医馆 (hospital) | 🏥 | 3 | 瘟疫抵抗 +Lv×10% | Lv1 "粗通医术" → Lv2 "妙手回春" → Lv3 "华佗再世" |
| 军械库 (armory) | ⚔️ | 5 | 武力上限 +Lv×20 | Lv1 "简陋兵器" → Lv3 "兵甲齐全" → Lv5 "武库充盈" |
| 粮仓 (granary) | 🌮 | 5 | 粮草上限 +Lv×20 | Lv1 "几筐余粮" → Lv3 "仓廪充实" → Lv5 "太仓满溢" |
| 官署 (office) | 🏛️ | 5 | 财富上限 +Lv×20 | Lv1 "草庐议事" → Lv3 "政令通达" → Lv5 "治世能臣" |
| 民房 (housing) | 🏠 | 5 | 人口容量 = 20 + Lv×40 | Lv1 "茅屋数间" → Lv3 "街巷繁荣" → Lv5 "万家灯火" |

## 3. 人口系统

| 参数 | 公式 | 当前值 |
|------|------|--------|
| 初始人口 | — | 50 |
| 人口上限 | 20 + 民房等级 × 40 | Lv1 = 60 |
| 每季粮消 | 人口 / 20 | — |
| 前端描述 | 同资源描述映射 | "人丁兴旺" / "人烟稀少" |
